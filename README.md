# Heart
Cardiovascular Risk Analytics is an end-to-end Python project utilizing clinical data to identify heart disease predictors. Features statistical correlation of 13 variables (cholesterol, ECG, chest pain), predictive modeling via Logistic Regression, and medical data visualization to enhance early diagnosis and patient care.
